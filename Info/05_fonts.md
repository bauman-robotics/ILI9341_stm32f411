# Способы получения шрифтов для дисплея ILI9341

## Общие сведения о шрифтах для TFT дисплеев

ILI9341 - цветной TFT дисплей с разрешением 320x240 пикселей. Для отображения текста используются bitmap шрифты, преобразованные в массивы байтов для встраиваемых систем.

## Форматы шрифтов

### 1. Bitmap шрифты
- Каждый символ представлен в виде массива пикселей
- Размеры: обычно 8x8, 8x16, 16x16, 16x24 пикселей
- Формат цвета: монохромный (1 бит на пиксель) или цветной (RGB565 для ILI9341)

### 2. Векторные шрифты
- Требуют больше ресурсов для рендеринга
- Подходят для больших размеров текста
- Реже используются в embedded системах из-за ограничений памяти

## Источники шрифтов

### 1. Библиотека Adafruit GFX
- Открытая библиотека с множеством шрифтов
- Включает базовые ASCII шрифты
- Репозиторий: https://github.com/adafruit/Adafruit-GFX-Library

### 2. STM32 community ресурсы
- Форумы и репозитории на GitHub
- Примеры проектов с ILI9341

### 3. Генераторы шрифтов
- **The Dot Factory** - онлайн инструмент для генерации bitmap шрифтов
- **Font Editor** - программы для создания пользовательских шрифтов
- **GLCD Font Creator** - специализированный инструмент для LCD дисплеев

## Шрифты с поддержкой кириллицы

### Доступные варианты

#### 1. Стандартные кириллические шрифты
- **Courier New** - моноширинный шрифт
- **Arial** - пропорциональный шрифт
- **Times New Roman** - с засечками

#### 2. Специализированные embedded шрифты
- **u8g2** - библиотека с поддержкой кириллицы
- **FreeFont** - свободные шрифты с кириллицей

#### 3. Пользовательские шрифты
Можно создать собственные шрифты с помощью:
- Конвертации TTF в bitmap
- Использования онлайн-конвертеров
- Программного создания массивов символов

### Примеры кода для кириллицы

```c
// Пример массива для символа 'А' (кириллица)

    0x00, 0x00, // ........

    0x0C, 0x00, // ...##...

    0x33, 0x00, // .##..##.
    0x33, 0x00, // .##..##.
    0x3F, 0x80, // .######
    0x33, 0x00, // .##..##.
    0x33, 0x00, // .##..##.
    0x00, 0x00, // ........
    0x00, 0x00, // ........
    0x00, 0x00, // ........
    0x00, 0x00, // ........
    0x00, 0x00, // ........
    0x00, 0x00  // ........
};
```

## Рекомендации по выбору шрифтов

### Для базового проекта
- Начать с 8x16 пикселей для читаемости
- Использовать моноширинные шрифты для простоты
- Ограничиться базовым набором символов

### Для кириллицы
- Проверить кодировку (UTF-8, CP1251, KOI8-R)
- Убедиться в наличии всех необходимых символов
- Рассмотреть размер занимаемой памяти

## Инструменты для создания шрифтов

### 1. Онлайн-конвертеры
- fontconvert (Adafruit)
- SquareLine Studio
- LVGL Font Converter

### 2. Десктопные приложения
- FontForge (бесплатный)
- BirdFont
- Glyphs (платный)

### 3. Специализированные утилиты
- STM32 Font Generator
- Embedded Font Generator

## Оптимизация

### Сжатие шрифтов
- RLE (Run-Length Encoding)
- Использование только необходимых символов
- Разделение на несколько шрифтов разных размеров

### Управление памятью
- Хранение шрифтов во flash памяти
- Кэширование часто используемых символов
- Динамическая загрузка шрифтов

## Заключение

Для ILI9341 рекомендуется использовать bitmap шрифты размером 8x16 или 16x24 пикселей. Для поддержки кириллицы подойдут стандартные шрифты типа Arial или Courier, сконвертированные в подходящий формат. Существуют готовые библиотеки и инструменты для генерации шрифтов с кириллицей.
