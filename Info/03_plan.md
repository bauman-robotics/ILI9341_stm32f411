# План работы по написанию Hello-World на дисплее ILI9341 с использованием FreeRTOS

## Цели
- Реализовать вывод текста "Hello World" на TFT-дисплей ILI9341 (320x240 пикселей) через интерфейс SPI
- Использовать операционную систему FreeRTOS для управления задачами
- Обеспечить корректную инициализацию дисплея и вывод текста

## Общий подход
1. Изучить документацию ILI9341 и ST7789V
2. Реализовать низкоуровневые функции для работы с SPI и GPIO
3. Написать драйвер для ILI9341 (инициализация, отправка команд и данных)
4. Реализовать функции для рисования текста (используя bitmap шрифты)
5. Создать FreeRTOS задачу для вывода Hello World
6. Протестировать на аппаратной платформе

## Риски и решения
